## Présentation du projet

Ce projet consiste en la création d'un **portfolio** en ligne, l'objectif est d'avoir un portfolio que je peux allimenter au fil du temps avec mes expériences, mes projets, mes compétences, afin de pouvoir facilement exposer mon parcours et mon évolution.

## Technologies utilisées

Pour la création de ce portfolio, j'ai utilisé les technologies React avec Vite, ainsi que Tailwind
